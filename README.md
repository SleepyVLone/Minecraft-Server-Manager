# Minecraft Server Manager

A command-line tool for managing Minecraft servers — written in C++.

> **Work in Progress** — this project is not functional as of yet.

## About

This is a personal portfolio project I'm building to learn C++. The goal is a tool that can start and stop a Minecraft server, back up world files and monitor player activity from the command line.

## Planned Features

- Start/stop server with configurable RAM settings
- Automated world backups with timestamps
- Player count monitoring via server logs
- Config file for easy customisation
